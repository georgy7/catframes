## Requirements

1. Bash
2. Python 3
3. ImageMagick
4. FFmpeg

## Usage

1. `extent_override.py` ensures, that all images in the current directory have the same resolution.
2. `images_to_h264.sh [--delete-images]` compresses them all to a single video file with 1 frame per second.
